## 📌 About the Project

**Planet Books** is a collaborative academic project focused on building a
virtual library system. This repository contains the frontend application,
responsible for delivering the user interface and interacting with the backend API.

The platform aims to provide an intuitive experience for users to explore books,
manage accounts, and (in future stages) allow administrators to control the system.


## ✨ Features

### 👤 Public/User Views
- 🏠 **Home** — landing page with featured content
- ℹ️ **About Us** — project and team information
- 📚 **Catalog** — browse and explore available books
- 🔐 **Login / Register** — user authentication system

### 🛠️ Admin Views *(coming soon)*
- 📊 Dashboard for system management
- 📖 Book management (CRUD)
- 👥 User management
- ⚙️ System configuration

## 🚀 Getting Started

### 📋 Prerequisites

- Node.js >= 18
- npm / yarn / pnpm
- Backend API running (Planet Books Backend)

### 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/planet-books-frontend.git

# Enter the project folder
cd planet-books-frontend

# Install dependencies
npm install

# Setup environment variables
cp .env.example .env
```
### ▶️ Run the Project
```bash
npm run dev
```
